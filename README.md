# 🌍 TravelFund

TravelFund es una aplicación Android desarrollada con Jetpack Compose para ayudar a planificar viajes, administrar presupuestos, registrar ahorros y gastos, y recibir recomendaciones inteligentes mediante Gemini AI.

---

## ✨ Características

- ✈️ Crear y administrar viajes.
- 💰 Registrar ahorros.
- 💸 Registrar gastos.
- 📂 Organizar el presupuesto por categorías.
- 📊 Estadísticas generales.
- 💱 Conversión de divisas.
- 🔔 Recordatorios y notificaciones.
- 🤖 Recomendaciones inteligentes usando Gemini AI.

---

## 📱 Tecnologías

## 🚀 Instalación

1. Clona el proyecto

```bash
git clone https://github.com/usuario/TravelFund.git
```

2. Agrega tu API Key de Gemini en `local.properties`

```properties
GEMINI_API_KEY=TU_API_KEY
```

3. Ejecuta el proyecto desde Android Studio.

---

## 🤖 Gemini AI

TravelFund utiliza Gemini para:

- recomendar actividades
- analizar el presupuesto
- sugerir nuevas categorías
- proponer consejos de ahorro
